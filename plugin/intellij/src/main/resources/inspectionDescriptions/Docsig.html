Reports docstring and signature mismatches found by the configured docsig executable for the current Python file.

Issues come from the same checker you run on the command line. The plugin runs it in the background (for example after save) and maps each message onto the reported source line as a weak warning.

Only files on the local file system are checked; the inspection skips non-local or synthetic roots. 


def f(a: int) -> None:
    """Missing params section."""
    pass

Set the docsig path and CLI-equivalent options under Settings | Tools | Docsig. The Test action there verifies the binary and minimum version before analysis runs.
